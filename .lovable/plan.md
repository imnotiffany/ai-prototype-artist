# 智能体权限管理 - 需求

## 入口
- 智能体管理详情页 `AgentDetail` 新增 Tab「权限管理」
- 项目内所有成员可进入并配置

## 权限模型
- 只配置一种角色：**User（使用者）** —— 可访问并对话该智能体
- 不涉及 Owner / Admin / Editor 等其他角色
- 新建智能体默认仅创建者可访问，其他人需手动添加

## 添加方式
1. **按工号添加**：支持单个/批量粘贴（逗号、空格、换行分隔），后端校验回填姓名、部门（无头像），非法工号行内报错
2. **按组织架构添加**：弹窗内含搜索框 + 部门树（中心/组织/部门/子部门），支持勾选整节点，选中节点下所有人员被展开加入名单

## 已授权名单
- 统一按**人员**展示：`工号 · 姓名 · 部门`
- 不保留部门条目形态（部门选择只是一次性批量加人）
- 支持搜索（工号/姓名/部门）、批量勾选移除
- 空状态：「仅创建者可访问，添加成员后他们即可使用该智能体」

## 保存
- 添加/移除即时生效，无草稿态与发布按钮
